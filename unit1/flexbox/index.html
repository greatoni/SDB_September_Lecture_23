This is my website using flex!

Accomplish

Horizontal or Vertical arrangement of elements.

Space distribution of elements.

Quick Notes

It is one-dimensional. This will either affect a row or column, but not both at the same time.

Useful for layout on nav bars, headers, and image galleries.

Place elements that are direct children but doesn't affect further children.

Flex Container

Our element with the "display:flex" style. This applies to the container and it's children.

Justify Content

Position content using the axis of the flex container.

  1. flex-start
  2. flex-end
  3. center
  4. space-between
  5. space-around
  6. space-evenly

Align Items

Similar to Justify Content, but works along the cross axis which is perpendicular to the primary axis (If the axis is row, Align items affects the column instead.).

  1. flex-start
  2. flex-end
  3. center
  4. stretch
  5. baseline
